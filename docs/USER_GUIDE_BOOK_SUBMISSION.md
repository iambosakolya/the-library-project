# User Guide: Submitting Books to the Catalog

## Introduction
Can't find a book you want to discuss in a club or event? You can now submit it to our catalog! After review, your book will be available for everyone to use in clubs and events.

## How to Submit a Book

### Step 1: Access the Submission Form
1. Log in to your account
2. Click on **"Submit Book"** in the user navigation menu
3. You'll see the book submission form

### Step 2: Quick Fill with ISBN (Recommended)
If you have the book's ISBN:

1. Enter the ISBN in the "Quick Fill with ISBN" field
   - Can be ISBN-10 (10 digits) or ISBN-13 (13 digits)
   - Example: `9780140283297` or `978-0-14-028329-7`

2. Click the **"Lookup"** button
3. If found, the form will auto-fill with:
   - Title
   - Author
   - Publisher
   - Publication date
   - Description
   - Page count
   - Language
   - Categories
   - Cover image

4. Review and edit any information as needed

### Step 3: Search Google Books (Alternative)
If you don't have the ISBN:

1. Enter the book title in the "Title" field
2. (Optional) Enter the author name in the "Author" field
3. Click **"Search Google Books"**
4. Select the correct book from the search results
5. The form will auto-fill with the book's information

### Step 4: Manual Entry
If the book isn't found via ISBN or Google Books:

1. **Title** (Required)
   - Enter the full book title
   - Include subtitle if applicable

2. **Author** (Required)
   - Start typing and select from suggestions
   - Or enter a new author name

3. **Description** (Required)
   - Provide a detailed description (10-2000 characters)
   - Include what makes this book special

4. **Categories/Genres** (Required)
   - Select at least one category
   - You can select multiple categories
   - Click categories to remove them

5. **Optional Fields**
   - ISBN-13: 13-digit ISBN
   - Publisher: Book publisher name
   - Published Date: When the book was published
   - Page Count: Number of pages
   - Language: Select from dropdown

6. **Cover Image** (Optional but recommended)
   - Click "Upload" to add a cover image
   - Supported formats: JPG, PNG, WebP
   - Maximum size: 4MB

### Step 5: Review and Submit
1. Review all entered information
2. Make sure at least one category is selected
3. Check the description is complete
4. Click **"Submit Book for Review"**
5. You'll receive a confirmation message

## Frequently Asked Questions

### Q: How long does review take?
A: Review times vary, but typically within 1-3 business days.

### Q: Why was my submission rejected?
A: Common reasons include:
- Book already exists in catalog
- Incomplete information
- Inappropriate content
- Copyright issues

### Q: Can I submit the same book again?
A: If rejected, you can resubmit with corrections. Check the rejection reason first.

### Q: What if the book isn't on Google Books?
A: No problem! You can manually enter all the information.

### Q: Do I need to know the ISBN?
A: No, but it makes the process much faster and more accurate.

### Q: Can I submit multiple books at once?
A: Currently, you must submit books one at a time.

### Q: What happens to approved books?
A: They become available in the catalog for everyone to use in clubs and events.

### Q: Can I get credit for my submission?
A: While your submission is recorded, books become part of the general catalog.
